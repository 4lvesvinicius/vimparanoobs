# Basico

## Instalação do VIM

Em sistemas baseados no Debian

```bash
$ sudo apt-get install vim
```

[Em outros sistemas operacionais.](http://www.vim.org/download.php "Página de Download oficial.")

## Aprender a usar o básico

Rode no Terminal

```
$ vimtutor
```
